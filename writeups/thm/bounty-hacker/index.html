THM - Bounty Hacker - WriteUP

Introduction

Bounty Hacker is one of the 'free rooms' implemented by TryHackMe. You have to find and obtain user & root flags.

  • Name : Bounty Hacker

  • Difficulty : Easy

  • OS : Linux

  • Type : Web-App Security

Let's dive into the challenge !
